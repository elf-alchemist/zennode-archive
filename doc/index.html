ZenNode - Possibly the fastest DOOM(tm) BSP builder yet.

Version 1.0 by Marc Rousseau



    ZenNode updates the BLOCKMAP, REJECT, and/or NODES/SEGS/SSECTOR resources of your .WAD file.  It supports DOOM / DOOM II / Heretic / and Hexen.
 
NEW
  • Real REJECT support!  ZenNode now builds a "perfect" REJECT resource.
FEATURES
  • The BSP tree generator is now user customizable!
  • Select from 3 different partitioning algorithms.
  • Control the parameters the algorithms use to select the partitions.
  • Indicate which LINEDEFs should be ignored by the BSP tree.
  • Select LINEDEFs/SECTORs that should not be split.
  • Specify whether or not a specific SECTORs must remain unique.
  • Rebuild the BLOCKMAP resource - compressed or uncompressed.
  • Rebuild the REJECT resource or insert a zero-filled resource.
  • Extract maps to a new PWAD file.
  • Merge multiple maps into a single .WAD file.
REQUIREMENTS
  • A machine capable of running DOOM - 386 or better with a few Megs of RAM
  • A supported operating:
    • Linux
    • A Win32 platform ( Windows NT or Windows 9x )
  • A .WAD file
